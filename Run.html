Run


MV 簡介及不專業解析：

　《 Run 》是花樣年華的其中一首主打歌， MV 故事與《 I Need You 》接連，繼續講述著七位不同青年的故事。

　與《 I Need You 》相比，《 Run 》可以明顯看出在這七位青年中，又可以分為四組。

　首先是 RM 與 V ，這兩個人似乎在所有人中是最叛逆的，兩人一起到處畫牆塗鴉，後來被警察抓起來，但 RM 後來好像做了什麼，使得 V 的表情有點驚訝，順帶一提， RM 在枕頭戰中，翻了一張撲克牌，上面是一隻蝴蝶，暗示著下一支影片 《 On Stage Prolong 》，還有 V 在 MV 中不斷有落水以及在水中掙扎的片段，也許是在暗示著什麼 ... 。

　接著是 Jimin 與 J-hope ，在《 I Need You 》中曾說過， J-hope 生病了，在這裡從他躺著的床旁邊吊著點滴、在馬路旁突然無力昏倒、在白天睡覺以及枕頭戰可以大概推斷是得了"嗜睡症"，而 Jimin 則是希望他能好起來並與他一同歡樂，在《 I Need You 》曾看到過 Jimin 在浴缸中泡水並燒著什麼，在這裡可以看到他在浴室與大家一同歡樂，感覺像是在慶生，而在影片的最後也終於知道，原來他燒的是一張大家曾經一起去海邊玩的照片，至於為什麼要燒掉並且泡著水痛哭呢？

　再來是 Suga 和 Jung Kook ，這兩個人在影片中有打架的場景，但仔細看看會發現其實是 Suga 好像情緒不穩定並抓狂，而 Jung Kook 不斷想阻止他停下才大打出手，當 Suga 把 Jung Kook 推倒後，他拿著椅子把鏡子打破，這個行為通常象徵著"打破虛假，回到現實"，此時 Jung Kook 也才意識到，曾經七人的歡樂時光，已經回不去了 ... 究竟發生了什麼事呢？而且在歌曲結束時， Jung Kook 對著開著車的 Jin 露出淡淡的微笑，令人不禁想起在《 I Need You 》時， Jung Kook 要被車撞的場景是否與 Jin 相關 ... 。

　最後是 Jin ,比起《 I Need You 》，在這裡可以看到他更多的場景，並且也可以推測他是七人中年紀最大的 ( 雖然在現實中也是 ) ，當他在堆牌塔時，好不容易建立起來，卻被 V 給一手拍掉，暗示著七人的分離將是從 V 開始，而從最後 Jimin 燒照片時看到照片中少一人，而這個人便是 Jin ，可以推測拍照的人就是 Jin 。

　在日文版 MV 中，也是加入了大量舞蹈元素，而從舞蹈概念中，是由 V 推動所有人起頭，更加確定七人的故事由他作為開始，而這裡看到 RM 不斷在電話亭的場景，那會是接了誰的電話呢？ Suga 的畫面則是在一地的破碎鏡子中，與原版相接他打碎鏡子，也許原來他才是活在虛假中不願接受現實的人，那是什麼樣的現實讓他無法接受呢？ Jimin 的畫面是他泡在裝水的魚缸裡面，時而縮著自己，時而扭曲掙扎，與他泡在浴缸裡的畫面相呼應，而且其中一幕可以看到魚缸上面寫著" Youth Is Not Coming Back 言外之意，也許是在說明，青春究竟是要接受魚缸中的現實，還是要不斷掙扎擺脫呢？ J-hope 與原本相比並無太多畫面，但是更確定的是他得了病是嗜睡症。 V 在這裡也是一樣有水的畫面，其中一個畫面他拿著噴漆噴出紅色，暗示著他可能有水難，那麼是怎麼遭遇的呢？Jung Kook 在這裡有個畫面值得思考，在原版《 I Need You 》中，原本是 Jin 手拿著百合花，在這裡變成 Jung Kook 拿並且捏碎，說明著他與 Jin 有什麼關聯，難道是車禍嗎？ Jin 在這裡雖然沒多少畫面，但有一幕是他望著牆壁上有黑色的蝴蝶，以及在這首歌的最後是以一個蝴蝶做結尾，暗示著接下來在《 On Stage Prolong 》會一點一點的慢慢呈現給大家究竟發生了什麼事情。
